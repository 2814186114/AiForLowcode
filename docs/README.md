# 低代码平台组件文档

## 简介

本文档为低代码平台的基础组件文档，采用Ant Design文档格式编写，包含所有可用的基础组件的详细说明、用法示例和API参考。

## 基础组件列表

| 组件名称 | 描述 | 文档链接 |
| --- | --- | --- |
| Button | 按钮组件，用于触发用户交互 | [Button文档](widgets/Button.md) |
| Input | 输入框组件，用于用户输入文本 | [Input文档](widgets/Input.md) |
| Select | 选择器组件，用于从列表中选择值 | [Select文档](widgets/Select.md) |
| Switch | 开关组件，用于开启/关闭状态切换 | [Switch文档](widgets/Switch.md) |
| Slider | 滑块组件，用于选择范围内的值 | [Slider文档](widgets/Slider.md) |
| DatePicker | 日期选择器，用于选择日期或日期时间 | [DatePicker文档](widgets/DatePicker.md) |
| Row | 行布局组件，用于创建水平布局 | [Row文档](widgets/Row.md) |
| Col | 列布局组件，用于创建垂直布局 | [Col文档](widgets/Col.md) |
| LoginForm | 登录表单组件，用于用户登录 | [LoginForm文档](widgets/LoginForm.md) |

## 文档结构

每个组件文档包含以下部分：

- **组件名称**：中英文名称
- **简介**：组件的核心功能描述
- **何时使用**：组件适用场景
- **代码演示**：不同使用方式的代码示例
- **API**：完整的属性、事件、方法列表
- **注意事项**：使用时的注意事项

## 使用说明

1. 在设计器中，从左侧组件库拖拽组件到画布
2. 在右侧属性面板中配置组件属性
3. 切换到预览模式查看组件效果
4. 导出代码或发布应用

## 组件使用规范

1. 每个组件应在其适用的场景下使用
2. 组件属性应根据实际需求合理配置
3. 复杂布局建议使用Row和Col组件配合实现
4. 表单类组件应注意数据绑定和验证
